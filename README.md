# API Project

A school project for the purpose of practicing how to implement API calls into a website using Javascript. Used the free Studio Ghibli API to display a grid of all the movies produced by Studio Ghibli; each movie title can also be clicked to pull up information about each movie. 

Link to the API used: https://ghibliapi.dev/#section/Getting-Started
